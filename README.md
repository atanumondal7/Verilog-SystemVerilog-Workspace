# Verilog-Workspace

## Coded in edaplayground.com

This repository contains all Verilog programs which I have written till date. It includes both the design code as well as a custom test-bench for testing the working logic of the program. 

### List of Programs:

1. AND Gate
2. OR Gate
3. 2 bit Multiplexer
4. 4 bit Adder
5. 4 bit Subtractor
6. 4 bit Counter
7. 4 bit ALU (Addition and Subtraction)
8. D Flip Flop
9. Traffic Light Simulator
10. Manual MUX (4 bit Structural)
11. Manual Full Adder (1 bit Structural)
12. Manual Ripple Carry Adder (4 bit Structural)
13. Manual MUX (4 bit BUS)
14. Manual Full Subtractor
15. Accumulator (Manual)
16. Self-Check Testbench
